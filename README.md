## Cheatsheet

### Search

#### Normal search

`/`

#### Search with visual mode

1) Go in visual mode with `v`
2) Select word
3) Search for occurrences with `*`

#### Search and replace multiple instances

1) Search word
2) Replace it with `cgn`
3) Go to next result with `n` or `N`
4) replace it with `.`
